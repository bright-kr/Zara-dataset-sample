# Zara 데이터셋 샘플

<h2>1001개의 Zara 제품으로 구성된 샘플 データセット</h2>

![Zara products dataset header](https://github.com/bright-kr/Zara-dataset-sample/blob/main/Zara-datasets.png)

1000개 이상의 레코드로 구성된 Zara 제품 データセット 샘플입니다. データセット은 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>Zara 제품 データセット에 포함된 일부 데이터 포인트:</h2>

* ```category_id```: 제품이 속한 카테고리를 식별하는 식별자  
* ```product_id```: 제품과 연관된 고유 식별자 또는 코드  
* ```product_name```: Zara 제품의 이름 또는 제목  
* ```price```: 제품의 정가 또는 원래 가격  
* ```currency```: 통화  
* ```colour_code```: 제품과 연관된 색상 코드  
* ```colour```: 제품에 제공되는 색상(들)  
* ```description```: 제품에 대한 텍스트 설명  
* ```size```: 제품에 제공되는 사이즈(들)  
* ```section```: 제품이 위치한 스토어 또는 웹사이트의 섹션  
* ```product_family```: 제품이 속한 패밀리 또는 카테고리  
* ```product_subfamily```: 제품이 속한 서브패밀리 또는 하위 카테고리  
* ```care```: 제품의 관리 또는 유지보수에 대한 정보  
* ```materials_description```: 제품에 사용된 소재에 대한 설명  
* ```materials```: 제품에 사용된 소재에 대한 정보  
* ```dimension```: 제품의 치수 또는 크기 측정값  
* ```low_on_stock```: 제품의 재고가 부족한지 여부를 나타냅니다(불리언: True/False)  
* ```availability```: 제품이 현재 구매 가능한지 여부를 나타냅니다(불리언: True/False)  
* ```image```: Zara 제품을 나타내는 이미지  
* ```sku```: 재고 관리 단위 또는 제품과 연관된 고유 식별자  
* ```url```: 제품 페이지로의 URL 또는 링크  

그 외에도 많은 항목이 포함됩니다.

이는 "Zara products"
データセット에서 파생된 샘플 서브셋이며, 해당 データセット에는 <b>999K개 이상의 레코드</b>가 포함됩니다.

사용 가능한 データセット 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, 또는 Parquet. 선택적으로 파일을 .gz로 압축할 수 있습니다</b>.

データセット 전달 방식 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 주기: <b>Once, Daily, Weekly, Monthly, Quarterly, 또는 Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 강화: <b>Based on request.</b>

<h2>Zara 제품 データセット의 활용 사례는 무엇입니까?</h2>

<h3>1. 가격 최적화</h3>
유사한 Zara 제품과 카테고리를 경쟁사와 비교하여 가격 전략 및 동적 가격 책정 모델을 수립합니다. 이를 통해 가격을 최적화하고 시장에서의 경쟁력을 확보하는 데 도움이 됩니다.

<h3>2. 제품 재고</h3>
Zara의 제품 라인업에서 재고 공백을 식별하고, 특정 품목에 대한 수요 증가를 추적하며, 소비자에게 트렌드로 떠오르는 제품을 파악하여 시장 수요에 맞게 재고 수준을 더 잘 정렬할 수 있습니다.

<h3>3. 시장 전략</h3>
Zara データセット을 활용하여 시장 전략 분석을 수행하고, 주요 트렌드와 고객 선호도를 파악하여 의사결정에 반영하고 제품 제공을 개선합니다.
